# 花札神経衰弱

> 花札の役・月の体系はそのままに、神経衰弱のゲーム性を組み合わせた新感覚カードゲーム。

---

## 遊び方

全48枚の花札を裏向きに並べ、順番に2枚ずつめくります。
**同じ月の札**なら回収できます。違う月の場合は裏向きに戻してターン終了。
場の札がなくなったらゲーム終了。
取得枚数ではなく、**集めた札で作れる「役」のポイント**で勝敗を決めます。

## 特徴

- **2〜4人**でプレイ（同じデバイスでのパスプレイ形式）
- 花札の役をすべて実装（五光・猪鹿蝶・赤短・青短・月見で一杯・花見で一杯 など）
- 役一覧をゲーム中いつでも確認可能
- ターン交代オーバーレイで次のプレイヤーへ明示的にバトンタッチ
- ビルド不要のシングルHTML・ブラウザのみで動作
