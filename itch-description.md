## ゲーム説明（日本語）

**花札と神経衰弱を組み合わせた、新感覚カードゲームです。**

全48枚の花札を裏向きに並べ、順番に2枚ずつめくっていきます。
同じ月の札なら回収できますが、違う月の場合は裏向きに戻してターン終了。

ポイントは——
**「たくさん取った人が勝ち」ではなく、「良い役を作った人が勝ち」。**

松に鶴、桜に幕、萩に猪……どのカードを取りに行くか、相手より先に狙いの月を揃えられるか。記憶力と役づくりの駆け引きをお楽しみください。

---

### 遊び方

1. 2〜4人のプレイヤーを設定してゲームスタート
2. 順番に2枚の札をめくる
3. 同じ月ならもらえる、違う月なら裏向きに戻す
4. 場の札がすべてなくなったらゲーム終了
5. 集めた札で作れる「役」のポイント合計で順位決定！

---

### 主な役（やく）

| 役名 | 点数 |
|------|------|
| 五光（光5枚） | 15点 |
| 猪鹿蝶 | 5点 |
| 赤短・青短 | 各5点 |

ゲーム中はいつでも「役一覧」ボタンで確認できます。

## Game Description (English)

**A Memory card game fused with Japanese Hanafuda.**

Flip two cards per turn. Match the same month to collect them.
But you don't win by collecting the most cards — you win by forming **Yaku (scoring combinations)**.

Which cards you collect matters. Strategy meets memory.

**Players:** 2–4 (pass-and-play on one device)
**Platform:** Browser (no install required)
